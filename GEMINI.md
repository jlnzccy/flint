# Gemini — start here

**Before doing anything in this repo:**

1. **Read [`plan.md`](plan.md) in full and follow it.** It is the single, current work plan. Do NOT
   create your own plan, do NOT re-scope, do NOT skip phases. If it conflicts with your own idea,
   `plan.md` wins. If it's genuinely blocking, stop and ask — don't improvise.
2. Read **[`AGENTS.md`](AGENTS.md)** for stack, design language, and product rules (terse copy,
   **no shame — never red, never ✗, no "behind" copy**, quiet attendance, no points/levels).
3. Backlog detail (anchors, accept criteria) is in **`task.md`**; completed history in
   **`task_done.md`** (read-only — don't add tasks there).

Workflow: do tasks in the listed order · `npx tsc --noEmit` must be clean before a phase is done ·
after each task mark it `[x]` in `task.md` with a one-line **Built:** note.
